# Mobile App Downloads

## Atmosfair Admin Mobile Application

This directory contains the Android APK file for the Atmosfair Sales Management mobile application.

### Instructions:

1. Place your Android APK file in this directory with the name: `atmosfair-admin.apk`
2. The download page will automatically serve this file to users
3. Update the version number in the download page when uploading new versions

### File Naming Convention:

- Main APK: `atmosfair-admin.apk`
- Versioned backups: `atmosfair-admin-v1.0.0.apk`, `atmosfair-admin-v1.0.1.apk`, etc.

### Current Version:

- Version: 1.0.0
- Last Updated: November 2025
- Platform: Android 8.0+

### Notes:

- Only Android applications are supported
- iOS is not available due to platform restrictions for enterprise apps
- This app is designed for Admin users and Sales Agents only
- Super Admin features are web-only

### Security:

- Ensure the APK is properly signed before deployment
- Keep the APK file secure and only distribute through official channels
- Update this README when releasing new versions
